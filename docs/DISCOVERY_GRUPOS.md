## Mapa Canônico: Grupo (ERP) → Categoria interna

> Regra de ouro: **grupo ESPECÍFICO do ERP é a fonte da verdade.** A heurística por nome/preço (`classifySmartphone`) só vale para o grupo genérico `CELULARES` e para itens sem grupo.

| Grupo (ERP, normalizado) | → Categoria | Confiança |
|---|---|---|
| `SUPER BONIFICADO` | SUPER BONIFICADO | 100% (ERP explícito) |
| `BONIFICADO` / `BONIFICADO LC` | BONIFICADO LC | 100% (ERP explícito) |
| `ANATEL` | ANATEL | 100% (ERP explícito) |
| `PELÍCULAS` / `PELICULAS` | PELÍCULA | 100% |
| `CASES` | CASES | 100% |
| `ACESSÓRIOS` | ACESSÓRIOS | 100% |
| `SERVIÇOS` | classifyServico → Proteção/Garantia/Assistência/Serviços | 95% (split por nome) |
| `CAIXAS JBL` / contém JBL/boombox/caixa de som | GERAL (não é celular) | alta |
| `GERAL` / `VENDAS GERAIS` | **DECISÃO**: GERAL (ERP) vs BONIFICADO LC (atual) | ⚠ negócio |
| `CELULARES` (genérico) | classifySmartphone(nome,preço) → BLC/SUPER/ANATEL | heurística |
| `∅` vazio + tipo=PEÇA DE MANUTENÇÃO | ASSISTÊNCIA TÉCNICA | alta |

## Precisão por mês (descoberta)
- **mar–mai:** ERP usa `CELULARES` genérico → split BLC/SUPER/ANATEL depende da heurística nome/preço (não é 100%).
- **jun em diante:** ERP migrou para `BONIFICADO`/`SUPER BONIFICADO`/`ANATEL` explícitos → split 100% preciso.
- **Caminho para 100%:** pedir ao lojista para usar sempre os grupos explícitos no Tenfront (já em curso desde junho).
